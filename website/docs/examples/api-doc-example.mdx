## Features

The ApiDoc component provides:

- **Clear function signatures** with TypeScript types
- **Parameter documentation** with optional/required indicators and default values
- **Return type documentation** with descriptions
- **Multiple examples** with syntax highlighting
- **Version information** with "since" badges
- **Deprecation warnings** for outdated functions
- **Responsive design** that works on all screen sizes
- **Dark mode support** that respects user preferences

## Usage in MDX Files

To use the ApiDoc component in your MDX files:

1. Import the component at the top of your MDX file:
```tsx
import ApiFunction, { ApiCategory } from '@site/src/components/ApiDoc';
```

2. Group related functions using `ApiCategory`:
```tsx
<ApiCategory title="Category Name" description="Category description">
  {/* Your ApiFunction components here */}
</ApiCategory>
```

3. Document each function using `ApiFunction`:
```tsx
<ApiFunction
  name="functionName"
  description="Function description"
  signature="function signature"
  params={[/* parameter definitions */]}
  returns={{/* return type info */}}
  examples={[/* code examples */]}
  since="1.0.0"
/>
```
