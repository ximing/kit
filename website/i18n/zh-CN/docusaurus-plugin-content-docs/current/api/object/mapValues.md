# `mapValues`

Creates an object with the same keys as object and values generated by running each own enumerable string keyed property through iteratee

## 参数

| 参数       | 类型  | 描述                                 |
| ---------- | ----- | ------------------------------------ |
| `obj`      | `any` | - The object to iterate over         |
| `iteratee` | `any` | - The function invoked per iteration |

## 返回值

- **类型**: `any`
- **描述**: Returns the new mapped object

## 示例

```typescript
* mapValues({ a: 1, b: 2 }, (value) => value * 2); // { a: 2, b: 4 }
```

```tsx live

            <strong>Example 1: Double all values</strong>
          </p>

          </p>
        </div>

          <p style={{ marginBottom: '10px' }}>

          </p>

      </div>
```
