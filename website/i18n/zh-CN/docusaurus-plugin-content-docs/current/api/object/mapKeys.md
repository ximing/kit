# `mapKeys`

Creates an object with the same values as object and keys generated by running each own enumerable string keyed property through iteratee

## 参数

| 参数       | 类型  | 描述                                 |
| ---------- | ----- | ------------------------------------ |
| `obj`      | `any` | - The object to iterate over         |
| `iteratee` | `any` | - The function invoked per iteration |

## 返回值

- **类型**: `any`
- **描述**: Returns the new mapped object

## 示例

```typescript
* mapKeys({ a: 1, b: 2 }, (value, key) => key + value); // { 'a1': 1, 'b2': 2 }
```

## 交互式示例

```tsx live

          <strong>Input Object:</strong> {JSON.stringify(obj)}

          <p style={{ marginBottom: '10px' }}>
            <strong>Example 1: Concatenate key and value</strong>
          </p>

            <strong>Result:</strong> {JSON.stringify(result1)}
          </p>
        </div>

          <p style={{ marginBottom: '10px' }}>

          </p>

        </div>
```
